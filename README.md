console-unzip
=============

Console program to unzip.

About
-----

Similar functionality to the minizip program, but supports unicode file name and the option to ignore dot files.

* libboost/1.57.0
* libz/1.28

```
 -p string : password
 -i string : input
 -o string : output
 -d : ignore dotfile
 -v : verbose
```
